Asset Management System

Complete User Documentation Guide

Getting Started

1. Registration as User

Once you open the link, you will see the Sign Up option located below the Login button.

To register as a user in the tool, please fill in the following details:

  • Name
  • Email Address
  • Phone Number
  • Password
  • Confirm Password

Complete the form to successfully create your account.

2. Login as User/Admin

To log in, please enter your credentials and click the Login button. Based on your role, you will be redirected to the appropriate dashboard.

Admin Portal

The Admin Portal includes several key components that help manage and monitor assets, users, and system operations efficiently.

Admin Portal Components

  • Dashboard
  • Asset Inventory
  • Add Asset
  • Alerts
  • Maintenance
  • Asset Requests
  • Reports
  • Users

Dashboard

The Dashboard offers a comprehensive overview for the admin, including:

Key Metrics
  • Total number of assets
  • Asset breakdown by Bond type and Type (Hardware/Software)
  • Status overview showing assets that are Available, In Use, or Under Maintenance
Visual Insights
  • Category distribution chart (pie or donut chart)
  • Bar chart illustrating monthly asset usage trends
  • Category-based filters for targeted insights
  • Detailed table showing asset distribution by provider

Asset Inventory

The Asset Inventory page shows a table with robust features that help manage assets effectively.

Asset Details Table Features

Dynamic Column Selection

Users can customize the table view by clicking "Select Columns" to show or hide specific data fields.

Export Functionality

Filtered asset data can be exported as an Excel file. You can filter by:

  • Type
  • Category
  • Status
  • Asset name (via search)
Interactive Features
  • Vendor Information: Click vendor names to view detailed vendor information in a popup
  • Assigned To Column: User icons show asset assignments with usage details
  • Actions Column: Edit, Notify, and Delete options for each asset
  • Print Today's Assets: Print list of assets added today for tagging and labeling

Add Asset

The Add Asset feature provides three distinct options for adding new assets to the system:

1. Hardware Assets

Manually add a hardware asset by entering specific details across several sections:

Basic Asset Details
  • Asset Bond Type, Name, Category, Number
  • Serial Number, Part Number, Asset Type, Status
  • Warehouse, Location, Provider, Classification
  • Model, Purchase Date, Cost, Warranty
  • Bond Expiry Date, Specification, Image, Supporting Documents
Vendor Details
  • Vendor Name, Contact, Notes
  • Add Custom Vendor button for seamless vendor management
Maintenance & Calibration Information
  • Last Maintenance Date and Interval
  • Calibration Date, Authority, Notes, Interval, Certificate

2. Software Assets

The Software Asset form includes the following required and optional fields:

Field Required Description
Asset Name Yes Name of the software
Software Version Yes Version number
Category Yes Software category selection
Asset Provider Yes Provider selection
Purchase Date Yes Date of purchase
Cost Yes Software cost
Renewal Date No License renewal date

3. Upload via Excel

For bulk asset entry, upload multiple assets using a formatted Excel file with these required columns:

Required Excel Columns

Asset Bond Type | Asset Name | Category | Model | Specification | Asset Number | Serial Number | Part Number | Asset Type | Asset Status | Item Floor | Storage Location | Rack Number | Storage Type | Location | Asset Provider | Classification | Purchase Date | Cost | Warranty | Vendor

Alerts

The Alerts section helps admins stay on top of critical asset-related events:

Alert Types
  • Maintenance Due
  • Bond Expiry
  • Software Renewals
  • Pending Requests
  • Warranty Expiry
Key Features
  • Filter by Alert Type
  • Show New Alerts with red background highlighting
  • Alert status tracking (new alerts become "seen" after viewing)

Maintenance

The Maintenance page provides a centralized view of all assets with scheduled or overdue maintenance.

Features
  • Asset list with maintenance status (Overdue/Upcoming)
  • Search functionality for specific assets
  • Filter by Category and Status
  • Proactive maintenance management

Asset Requests

This section provides comprehensive views of both Asset Requests and Asset Returns with easy navigation between tabs.

1. Asset Requests Tab

Information Description
Asset Name & Number Identification details
Requested By User who made the request
From/To Date Usage period
Status & Stage Current status with full lifecycle history
Actions Approve, Reject, Issue options

2. Asset Returns Tab

Similar table structure showing returned asset details with Accept, Received, and Closed action options.

Reports

The Reports tab provides powerful tools for generating, analyzing, and customizing asset-related reports.

Report Types

1. Asset Data

Basic details of all assets including name, type, number, status, and assigned user information.

2. Cost Analysis

Comprehensive cost overview segmented by Provider and Asset Classification for understanding cost distribution.

3. Idle Days Report

Displays idle days count for each asset to monitor underutilized resources and improve allocation efficiency.

Customize Reports Feature

Access the Custom Layout tab to design your own report layout with these tools:

  • Layout Selection: Choose between Free Layout and Title Layout modes
  • Chart Panel: Drag and drop pre-built chart widgets onto the canvas
  • Customization: Resize charts and modify colors, labels, and formatting
  • Configuration Management: Download and load saved layout configurations
  • Export: Download customized reports as PDFs

Users Tab

The Users tab provides an overview of all registered users in the system.

Key Features
  • User listing with name, email, role, and contact information
  • Admin ability to delete users for access control
  • User database maintenance and security management

Asset Inventory

Browse available assets and make new requests through a comprehensive asset view.

Key Features

Asset Details Table
  • Dynamic Column Selection: Customize table view by showing/hiding specific data fields
  • Export Functionality: Export filtered data as Excel files
  • Vendor Information: Click vendor names for detailed information popups

Asset Requests

Comprehensive view of both Asset Requests and Asset Returns with easy navigation between tabs.

Navigation & Filtering

Navigation Options
  • Asset Requests Tab: View and manage outgoing requests
  • Asset Returns Tab: View and manage asset returns
Filtering Capabilities
  • Stage Filter: Filter by request/return stages
  • Date Range Filter: Filter by specific date ranges
  • Stage Date: Filter by specific stage dates

1. Asset Requests Tab

Column Description
Asset Name Name of the requested asset
Requested By Your name (the requester)
From/To Date Asset usage period
Purpose Reason for requesting the asset
Status Color-coded current status (Approved, Pending, Rejected, Completed)
Requested At Submission timestamp
Actions Available actions based on status (Received button for approved requests)
Request Status Flow

Requested → Pending → Approved/Rejected → Issued → Received

2. Asset Returns Tab

Column Description
Asset Name Name of the returned asset
Returned By Your name (the returner)
Issue Date Original asset issue date
Return Date Date when asset was returned
Purpose Original request purpose
Status Color-coded return status (Surrendered, Return Received, Return Accepted)
Return Status Flow

Returned → Return Received → Surrendered → Closed
